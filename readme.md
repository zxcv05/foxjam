# Foxy Flips
> Flip coin, Win money, Get foxes

Submission for [Mini Jam 176: Foxes](https://itch.io/jam/mini-jam-176-foxes)<br/>

## Building
- Built with zig 0.14.0-dev.2634+b36ea592b

- `zig build run`
